## How to run

To get started:

* install all project dependencies with `npm install`

  Notable dependencies:
```js
    "react-modal": "^3.4.4",

    "styled-components": "^3.2.6",
    
    "uuid": "^3.2.1"
```

* start the development server with `npm start`
